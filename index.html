I have been learning how to build apps using Dart and the Flutter framework. Flutter is a cross platform framework developed by Google which uses the Skia rendering engine. The great thing about cross platform frameworks is that you can deploy the same codebase to iOS, Android, web, Windows, Mac and Linux. Right now Flutter only fully supports iOS and Android but Web is in beta. So far I have been building a simple calendar application which uses a local database to store information.
